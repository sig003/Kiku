# KIKU — Play 스토어 등록 정보 (초안)

> Play Console > 기본 스토어 등록정보에 넣을 문구. 필요시 다듬어 사용.

## 앱 이름 (최대 30자)
```
KIKU - 일본어 청해(JLPT)
```

## 짧은 설명 (최대 80자)
```
화면 안 봐도 듣는 JLPT 청해 훈련. N4·N3 문장과 대화를 반복 청취하세요.
```

## 자세한 설명 (최대 4000자)
```
🎧 화면을 보지 않고 "듣기만" 하는 JLPT 일본어 청해 훈련 앱

라디오·팟캐스트처럼, 화면을 끄고 주머니에 넣은 채로 일본어 문장을 반복해서 들으며 귀를 트이게 합니다. 출퇴근·산책·집안일 하면서 흘려듣기 딱 좋아요.

■ 이렇게 들려줍니다 (문장 드릴)
일본어 3회 → (해석할 시간) → 한국어 뜻 → 일본어 한 번 더 → 핵심 단어
한 문장을 충분히 곱씹도록 간격을 두고 반복합니다.

■ 대화 모드
A와 B가 한 문장씩 주고받는 실전 대화. 남녀 목소리가 번갈아 나와 실제 회화처럼 들립니다.

■ 주요 기능
• 백그라운드 재생 — 화면 꺼도, 다른 앱 써도 계속 재생
• 이어폰·잠금화면 컨트롤 — 재생/정지/이전/다음
• N4·N3 난이도, 회사생활·여행·생활회화·대화 카테고리
• 전체 랜덤 — 모든 문장에서 무작위로 섞어 듣기
• 한국어 해석 숨기기(섀도잉), 재생 속도 조절(0.8/1.0/1.2)
• 이어듣기 — 앱을 닫아도 마지막 위치부터

■ 이런 분께
• 유튜브 "일본어 100문장 듣기"로 공부하다 아쉬웠던 분
• 내 수준(N4·N3)에 맞는 문장만, 원하는 방식으로 반복하고 싶은 분
• 화면 못 보는 상황(운전·운동·집안일)에서 귀로 공부하고 싶은 분

인터넷 없이도 동작하며(기기 내장 음성 사용), 회원가입이 없습니다.
```
