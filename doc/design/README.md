# Handoff: KIKU — 청해 앱 UI 리디자인 (아이팟/오디오 플레이어 감성)

## Overview
JLPT 청해(듣기) 학습 앱 **KIKU**의 메인 UI 리디자인. 오디오 플레이어(아이팟 / Apple Music) 감성으로,
다크 배경 + 골드 액센트로 "문장을 듣는" 경험을 강조한다. 두 화면을 다룬다: **라이브러리**(컬렉션 목록)와
**나우플레잉**(문장 드릴/재생).

## About the Design Files
이 번들의 `ui-redesign.html`은 **HTML로 만든 디자인 레퍼런스**다 — 최종 룩앤필과 동작을 보여주는 목업이며,
그대로 가져다 쓰는 프로덕션 코드가 아니다. 목표는 이 디자인을 **KIKU 안드로이드 앱의 기존 환경(Jetpack Compose 가정)
으로 재현**하는 것이다. 앱이 Compose가 아니라 View/XML이면 동일 스펙을 그 패턴으로 옮기면 된다.
HTML/CSS 자체를 앱에 이식하지 말 것.

## Fidelity
**High-fidelity (hifi)** — 색/타이포/간격/인터랙션이 확정된 픽셀 목업이다. 아래 토큰과 스펙 값을 그대로 적용해
픽셀에 가깝게 재현할 것.

### Typography
- **일본어**: Noto Sans JP (앱에선 시스템 일본어 폰트 또는 번들). Weights 400/500/700/900.
- **한국어/UI**: Pretendard (또는 앱 기본 Sans). Weights 400–800.
- 스케일: 대제목 22–26 / 문장(일본어) 26 / 본문 15–16 / 라벨 12–14 / 메타·캡션 11–12
- 레터스페이싱: 제목 -0.02em, 일본어 문장 +0.01em, 대문자 라벨(RANDOM 등) +0.08~0.14em

### Spacing / Radius / Shadow
- 4px 베이스. 화면 좌우 패딩 20–24px, 카드 내부 패딩 16–20px.
- Radius: 칩/버튼 pill(999), 카드 16–22, 아트 타일 14, 미니바 18.
- 그림자: 카드/디스크 `0 20px 50px rgba(0,0,0,.55)`, 골드 재생버튼 `0 10px 26px rgba(243,193,78,.36)`.
- 모션: 120–180ms ease-out. 재생 버튼 누름 `scale(.96)`. 바운스 없음.

## Screens / Views

### 1) 라이브러리 (Library)
**Purpose**: 학습 컬렉션을 고르고, 전체 랜덤으로 바로 시작. 재생 중이면 하단 미니 플레이어로 이어짐.

**Layout** (세로 스크롤):
1. **상단 브랜드 바** — 좌측 골드 마스크 아이콘(40×40, radius 11, 배경 `#F3C14E`, 중앙 `聞` 900 24px `#0F1115`)
   + 우측 텍스트 `kiku`(Noto Sans JP 900, 22px) / 하단 `JLPT LISTENING`(11px, `#6B7280`, letter-spacing .14em).
2. **히어로 카드 "전체 랜덤"** — radius 22, 패딩 20, 배경 `radial-gradient(120% 140% at 15% 0%, #3A2E12 0%, #171821 55%)`,
   보더 `1px rgba(243,193,78,.28)`. 우하단에 큰 셔플 글리프 `rgba(243,193,78,.10)` 장식.
   - 라벨 `◇ 오늘의 듣기`(골드 11px 700 대문자 letter-spacing .08em)
   - 제목 `전체 랜덤`(23px 800)
   - 설명 `모든 문장에서 무작위로. 열 때마다 새로 섞여요.`(13px `#9BA3B0`, max-width 78%)
   - CTA 필 버튼 `▶ 30문장 시작` (배경 골드, 텍스트 `#0F1115` 800 14px, radius 999, 패딩 10×18)
3. **섹션 라벨** `컬렉션 · N4` (12px 700 `#6B7280` 대문자 letter-spacing .1em)
4. **컬렉션 행 리스트** (행 사이 1px 구분선 `rgba(255,255,255,.08)`):
   - 각 행: 아트 타일(56×56, radius 14, 위 색 그라데이션, 중앙 카테고리 한자 900 24px `#FFF`, 우하단 `N4` 8px)
     + 텍스트(카테고리 라벨 11px 700 `#6B7280` / 제목 16px 700 / 서브 `100문장 · 드릴` 12px `#9BA3B0`)
     + 우측 원형 재생 아이콘(34×34, radius 50%, 보더 1.5px `rgba(255,255,255,.08)`, `▶` `#9BA3B0`)
   - 데이터: `회사생활 / 회사·업무 100문장`(会社), `여행 / 여행 100문장`(旅), `생활회화 / 생활회화 100문장`(会話)
5. **하단 미니 플레이어** (고정, floating) — left/right 14, bottom 14, height 60, radius 18,
   배경 `rgba(28,33,44,.86)` + `backdrop-filter: blur(16px)`, 보더 1px, 그림자 `0 12px 30px rgba(0,0,0,.5)`.
   - 좌측 40×40 아트(인디고 그라데이션, 골드 `聞`) + 제목(일본어 13px) / 서브 `전체 랜덤 · 1/30`(11px 뮤트)
   - 우측 40×40 골드 원형 일시정지 버튼(`❚❚`, `#0F1115`)

### 2) 나우플레잉 (Now Playing)
**Purpose**: 문장 하나를 듣고, 뜻/어휘 확인, 이전·다음·다시·속도 조절.

**Layout** (세로, 배경 `radial-gradient(120% 80% at 50% 0%, #141A28 0%, #0F1115 60%)`):
1. **상단 바** — 좌측 뒤로(38×38 원형 `#191E28`, `‹`) / 중앙 라벨 상단 `RANDOM`(11px 700 `#6B7280` .08em) + 하단 `전체 랜덤 30문장`(14px 700) / 우측 더보기(`⋯` 뮤트)
2. **디스크(아트)** — 230×230 원, `radial-gradient(circle at 38% 34%, #232A38, #0E1118 70%)`,
   그림자 `0 20px 50px rgba(0,0,0,.55)` + inset 1px `rgba(255,255,255,.05)`.
   동심원 링 3개(border `1px rgba(243,193,78,.14)`, inset 0 / 26 / 52). 중앙 `聞`(Noto Sans JP 900, 96px, `#FFF`).
   우상단 `1 / 30` 뱃지(11px `#6B7280`). *(디스크 = 아이팟/턴테이블 오마주. 중앙 구멍 없음.)*
3. **일본어 문장** — 26px 700, 중앙, line-height 1.4. 예: `この電車は混んでいますね。`
4. **한국어 뜻** — 15px `#9BA3B0`, 중앙. 예: `이 전철은 붐비네요.`
5. **`한국어 숨기기` 토글** — 골드 12px 600 텍스트 버튼. 누르면 뜻 visibility 토글 + 라벨 `한국어 보기`.
6. **어휘 칩** — 중앙 정렬, gap 8. 칩: 일본어 12.5px `#CBD2DC`, 배경 `#191E28`, 보더 1px, radius 999, 패딩 6×12.
   예: `電車 · 전철`, `混む · 붐비다`
7. **스크러버** — 트랙 height 4 radius 2 배경 `#1F2530`; fill(현재 14%) 골드; dot 11×11 골드 + 헤일로 `0 0 0 4px rgba(243,193,78,.18)`. 아래 시간 `0:00` / `0:04`(11px `#6B7280`).
8. **컨트롤** — 중앙 정렬 gap 26: 이전(52 원형 `#191E28` `⏮`), **재생(74 원형 골드 `▶`, 그림자)**, 다음(52 `⏭`). 재생 누르면 `❚❚` 토글.
9. **속도 세그먼트** — 라벨 `속도` + 세그(배경 `#191E28`, radius 999, 패딩 3): `0.8 / 1.0 / 1.2`, 선택 시 배경 골드·텍스트 `#0F1115`. 기본 1.0.

## Interactions & Behavior
- 라이브러리 행/히어로 탭 → 나우플레잉으로 이동 (선택 컬렉션 로드).
- 미니 플레이어 탭 → 나우플레잉 확장. 미니바 재생버튼 → 재생/일시정지 토글.
- 나우플레잉: 재생 버튼 = play/pause 토글, 이전/다음 = 문장 이동(스크러버·`n/30`·문장·뜻·어휘 갱신), 다시 = 현재 문장 처음부터.
- 속도 세그먼트 = TTS/오디오 배속(0.8/1.0/1.2) 즉시 적용.
- `한국어 숨기기` = 뜻 표시/숨김 토글(섀도잉 학습용). 상태는 세션 유지 권장.
- 오디오 진행에 따라 스크러버 fill/시간 실시간 업데이트.

## State Management
- `collections: List<Collection>` (id, category, title, count, artColors, kanji)
- `session`: 현재 컬렉션, 문장 리스트(랜덤이면 열 때 셔플), `index`(0..n-1), `total`
- `player`: `isPlaying`, `positionMs`, `durationMs`, `speed(0.8|1.0|1.2)`
- `ui`: `showKorean(Boolean)`
- 전이: 행 탭→세션 생성+index0; 다음/이전→index±1(경계 처리); 재생 완료→자동 다음(옵션).

## Assets
- 앱 아이콘/마스크 심볼은 별도 폴더 `kiku/master/`(1024 투명 PNG, 배경 `#0F1115`) 참조.
- 화면 내 한자(聞/会社/旅/会話)는 이미지가 아니라 **폰트 글리프**로 렌더 — 별도 에셋 불필요.
- 아이콘(▶ ❚❚ ⏮ ⏭ ‹ ⋯ 등)은 앱의 아이콘 세트(Material Symbols 등)로 대체 권장. 골드/뮤트 색만 맞출 것.
